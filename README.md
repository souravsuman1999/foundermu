## Founders' Union (Next.js)

### Getting started

```bash
npm install
npm run dev
```

### Theme customization

- Edit **`src/app/globals.css`** for global colors, spacing, radius, and other tokens.
- The accent color is Tailwind color **`fu`**:
  - `rgb(218 255 1 / <alpha-value>)`
- Change it in **`tailwind.config.ts`** if you ever want a new accent.

### Page content

- Main page: **`src/app/page.tsx`**
- Components: **`src/components/*`**
